# Bill of Lading Request Generator

A streamlined Flutter application for generating Bill of Lading (B/L) draft requests for shipping and Letter of Credit (LC) issuance.

## Features
- **Customizable Templates:** Enter shipment details (Shipper, Cargo, Quantity) to automatically generate professional request emails.
- **Real-time Preview:** See the email draft update as you type.
- **Responsive Design:** Works seamlessly on mobile, tablet, and desktop web browsers.
- **One-Click Copy:** Easily copy the generated text to paste into your email client.

## Technical Stack
- Framework: Flutter
- Language: Dart
- Platforms: iOS, Android, Web, Desktop

## Setup
1. Ensure Flutter is installed on your system.
2. Run `flutter pub get` to fetch dependencies.
3. Run `flutter run` to launch the application.
